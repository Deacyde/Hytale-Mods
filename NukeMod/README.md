# Nuke Mod — Hytale Asset Pack v1.0.0

Adds a **throwable Nuke Block** to Hytale — a massive explosion weapon.

> ⚠️ **This is the original nuke-only version.** The updated **TNTBlockMod** includes both TNT Block and Nuke Block with improved configs. Use that instead unless you specifically want just the nuke.

---

## Item

| Item ID | Name | Quality | Description |
|---------|------|---------|-------------|
| `Nuke_Block` | Nuke Block | Epic | Throwable nuke — massive explosion on impact |

---

## How to Use

1. Get the **Nuke Block** from the creative inventory
2. **Throw it** — hold and release / drop from hand
3. It flies through the air and **explodes on impact**
4. The explosion destroys terrain and damages all nearby entities in a large radius

---

## Installation

1. Copy the mod zip file to:
   ```
   ~/.var/app/com.hypixel.HytaleLauncher/data/Hytale/UserData/Mods/
   ```
2. Launch the world — the Nuke Block appears in the creative inventory

> **Asset pack only** — no Java plugin required.

---

## Notes

- Requires Hytale build `>= 2026.02.19`
- **Superseded by TNTBlockMod** — which adds both TNT Block + Nuke Block with refined explosion configs
- Do not load both NukeMod and TNTBlockMod at the same time (duplicate Nuke_Block item)
